# Basic Template

A simple, clean starting point for learning Duso. Covers:

- **Variables** - Numbers, strings, booleans
- **Arrays** - Lists of values with iteration
- **Objects** - Key-value data structures
- **Functions** - Reusable code blocks
- **String concatenation** - Building strings with `+`
- **Control flow** - `for` loops and conditionals

## Running

```bash
duso basic.du
```
