# Hello Template

A beautifully formatted "Hello World" that showcases:

- **String formatting** - Multiline strings with `"""`
- **Modules** - Loading the markdown stdlib module
- **Environment variables** - Checking `$NOCOLOR` for user preferences
- **Conditionals** - Branching logic with `if/else`
- **ANSI output** - Pretty colored terminal output

## Running

```bash
duso hello.du
```

Disable colors:
```bash
NOCOLOR=1 duso hello.du
```
